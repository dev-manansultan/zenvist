# Glossary

## Active Earning Session
A verified desktop session currently available to run assigned visits and earn credits.

## Demand Queue
Ordered list of campaigns/websites waiting to receive traffic.

## Supply Pool
Set of currently active contributor sessions that can be assigned demand targets.

## Earning Rate (earn_rate_ppm)
Credits earned per valid minute by contributor sessions.

## Spending Rate (spend_rate_ppm)
Credits charged per delivered minute to receiver campaigns.

## Country Targeting
Campaign rule restricting delivery to signed-up contributors from selected countries.

## One-by-One Dispatch
Assignment policy where each active supply session gets only one target at a time.

## Wallet Reservation
Pre-run hold on receiver credits to prevent overspending before assignment starts.

## Settlement
Final ledger operation that debits receiver and credits contributor after actual run duration.

## Live View
Near real-time desktop panel showing active browser preview and event timeline.

## Quality Multiplier
Factor applied to earnings based on session quality and anti-abuse checks.
